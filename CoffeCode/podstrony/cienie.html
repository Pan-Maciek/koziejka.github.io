Cienie tekstu

Jak dodać cień tekstu?

            text-shadow:-5px 5px 5px red;

TEST test test

Pierwsza wartość odpowiada za przesunięcie cienia w poziomie, druga w pionie, trzecia rozmycie a ostatnia kolor cienia.

Cienie elementów

Jak dodać cień do elementu?

"Odpowiedź jest bardzo łatwa" tak samo jak do tekstu jednak zamiast właściwości text-shadow należy użyć właściwości box-shadow

            box-shadow:1px 1px 5px gray;

TEST test test

Warto powiedzieć również, że jeden element tak jak i tekst może mieć kilka cieni co może w tym przypadku nie wyglądać za ładnie ale może kiedyś przyda Ci się ta informacja.

            box-shadow:1px 1px 5px red, -1px -1px 50px gold;

TEST test test

Oto ładniejsze wykorzystanie tej właściwośći

            box-shadow:2px 2px 5px gray;

TEST test test

Jak widzimy nasz cień sprawia wrażenie żucanego pod kątem a my chcemy uzyskać takie efekt jakby światło świeciło pionowo co logicznie nie ma sensu gdyż nie powstał by cień ale wygląda świetnie.

            box-shadow:2px 5px 2px gray, -1px -1px 2px gray;

TEST test test
